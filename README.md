## Repository Introduction

[Langchain-Chatchat]: https://github.com/chatchat-space/Langchain-Chatchat

is an intelligent dialogue platform developed based on the Langchain framework。This product provides an out-of-the-box langchain-chatchat based on the Huawei Cloud EulerOS 2.0 64-bit system of Kunpeng servers.

## Core Features

- Local Knowledge Enhancement: Supports private document import and vectorized storage, combined with a large language model to achieve accurate local knowledge question answering, improving the accuracy and credibility of answers
- Multi model adaptation: * * Compatible with mainstream open source models (such as ChatGLM, gemma, deepseek, etc.), supports custom model integration, and flexibly adapts to different scene requirements

The open-source image product Langchain Chatcat Intelligent Dialogue System provided by this project has pre-installed the 0.3.1.3 version of langchain-chatchat and its related runtime environment, and provides deployment templates. Come and refer to the usage guide to easily start an efficient "out-of-the-box" experience!

> **System requirements are as follows:**
>
> - CPU: 4 vCPUs or higher
> - RAM: 16GB or larger
> - Disk: At least 40GB
